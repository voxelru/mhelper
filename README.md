# mhelper — задачи (Flask + SQLite)

## Запуск (Windows / PowerShell)

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Откройте в браузере `http://127.0.0.1:5001`.

## Вкладки

### Доска

Основной рабочий экран в формате канбан: три секции **Планируется**, **В работе** и **Завершено**. Задачи перетаскиваются между секциями (drag&drop), статус сохраняется в SQLite. Вверху — быстрая форма создания задачи и фильтр по группе (включая пункт **Без группы**). Можно добавлять новые группы (предметные области). На карточке задачи видны приоритет, группа, исполнитель, срок, ссылка и счётчики комментариев.

### Все задачи

Табличный список всех задач, в том числе из архива. Фильтры: название, статус, приоритет, группа (с пунктом **Без группы**), исполнитель, диапазон сроков. Сортировка по колонкам. В таблице показан последний комментарий к задаче. Из строки можно открыть карточку задачи или перейти к редактированию.

### Комментарии

Список открытых (нерешённых) комментариев по задачам, которые не в архиве. Те же фильтры по задаче, что и на вкладке «Все задачи». Комментарий можно отметить выполненным прямо в таблице. Для каждой строки доступны переход к задаче и редактирование.

## Карточка задачи

Страница отдельной задачи: все поля (приоритет, группа, исполнитель, срок, ссылка, описание), комментарии (новые сверху), добавление и редактирование комментариев, отметка «решено».

## Общее

- Поля задачи: название, описание, ссылка, исполнитель, срок, приоритет (Pr0–Pr4), группа, статус (включая архив)
- Комментарии сохраняются с датой и временем
- Данные хранятся в SQLite
