# mhelper — задачи (Flask + SQLite)

## Запуск (Windows / PowerShell)

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Откройте в браузере `http://127.0.0.1:5000`.

## Что реализовано

- 3 вертикальные секции: **Планируется**, **В работе**, **Завершено**
- Создание задач в верхней секции
- Перетаскивание задач между секциями (drag&drop) с сохранением в SQLite
- Группы (предметные области) и фильтр по группе
- Поля задачи: название, ссылка, исполнитель, срок
- Комментарии к задаче, сохраняются с датой/временем
